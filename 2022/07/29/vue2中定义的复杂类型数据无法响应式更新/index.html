vue2中定义的复杂类型数据无法响应式更新.
2022-07-29

答 : 涉及到复杂类型的变量, 使用 官方自带的 $set, $delete 工具进行增删改操作.

知识点

$set/$delete

在 Vue2 中，修改某一些数据，视图是不能及时重新渲染的。原因就是 Vue2 中的数据响应式是利用 object definedProperty()实现的，它是无法深层监听数据的变化的。因此，Vue2 就提供了一些变异的方法，比如$set $delete

案例
const editableData = {};

// $set(对象, 对象属性, 属性值);
$set(editableData, id, item);

// $delete(对象, 对象属性);
$delete(editableData, id);
