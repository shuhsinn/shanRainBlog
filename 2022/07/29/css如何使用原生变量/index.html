css如何使用原生变量
2022-07-29

答 : 需要使用css3新增的一个伪类选择器root，将定义好的变量(均以 – 声明:例如 –color)挂载到root选择器上,然后在其他地方使用var()语法引用

知识点

:root

root是一个伪类，表示文档根元素，所有主流浏览器均支持 :root 选择器，除了 IE8 及更早的版本。在:root中声明相当于全局属性，只要当前页面引用了:root 变量所在文件，都可以使用var()来引用。

var()

var()函数可以代替元素中任何属性中的值的任何部分。var()函数不能作为属性名、选择器或者其他除了属性值之外的值。(var()函数无法计算 scss/less 等需要编译才能正常使用的变量)

案例
//定义
:root {
    --el-dark-value: 10%;
    --el-font-size-base: 25px;
    --el-color-primary-lighter: #72a1ed;
    --el-color-primary-lightest: #9abcf2;
    --el-color-secondary: #ebedf0;
}

//使用时
font-size: var(--el-font-size-base);
